### Необходимо сверстать три страницы по макетам: directories.html, directory.html, entry.html (html, css)
### Страницы должны формироваться динамически на основе данных, полученных из соответствующих json-файлов (папка json) (javascript)
### На странице directory.html должен работать простеший текстовый фильтр (javascript)
### На странице entry.html должен работать инпут даты (javascript)

### Можно использовать jquery, jquery ui и любые другие фреймворки если необходимо
